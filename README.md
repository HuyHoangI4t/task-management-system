# Fullstack Starter Project

## Cấu trúc

- `server`: API Node.js + Express
- `client`: Giao diện React + Vite

## Chạy project

### 1) Server

1. Mở terminal tại thư mục `server`
2. Cài package: `npm install`
3. Tạo file `.env` từ `.env.example`
4. Chạy: `npm run dev`

### 2) Client

1. Mở terminal tại thư mục `client`
2. Cài package: `npm install`
3. Chạy: `npm run dev`
